# Bank Loan Analytics Dashboard
## Project Overview
This Power BI project analyzes bank loan data to monitor loan performance, funding distribution, repayments, interest rates, and borrower risk profiles.
## Objectives
- Track total loan applications
- Analyze funded amount and repayments
- Identify Good vs Bad loans
- Monitor interest rates and DTI ratios
- Support data-driven lending decisions
## Dashboard KPIs
- Total Loan Applications: 313
- Total Fund Amount: 6.3M
- Total Amount Received: 7.3M
- Average Interest Rate: 21.40%
- Average DTI Ratio: 14.06%
## Key Insights
### Good Loans
- Applications: 198
- Funded Amount: 4.2M
- Amount Received: 5.8M
- Contribution: 63.26%
### Bad Loans
- Applications: 115
- Funded Amount: 2.2M
- Amount Received: 1.5M
### Loan Status Analysis
- Fully Paid loans generated the highest repayment amount.
- Current loans showed the lowest application volume.
- Interest rates were highest among current loans.
## Tools Used
- Power BI
- Microsoft Excel
- DAX
- Data Modeling
- Data Visualization
## Business Impact
This dashboard helps financial institutions:
- Reduce lending risk
- Monitor loan performance
- Improve collection strategies
- Make data-driven lending decisions
